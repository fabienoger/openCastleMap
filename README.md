# Open Castle Map

Map that lists the castles in France.

#Project

This projects using MeteorJS, Materializecss and Leaflet.

#Usage

You just need to run the following command line :
`meteor`

## Packages

* materialize:materialize
* iron:router
* bevanhunt:leaflet
* ejson
* herrhelms:meteor-leaflet-markercluster
* fourseven:scss
* mdg:geolocation
